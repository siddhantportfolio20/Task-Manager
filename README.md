# 📋 Task Manager

A responsive **Task Manager** web application built using **HTML, CSS, and JavaScript**. The application allows users to efficiently organize their daily tasks through a clean and intuitive interface. Tasks are stored using the **Local Storage API**, ensuring they remain available even after refreshing the browser.

---

## ✨ Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete all tasks
* 💾 Store tasks using Local Storage
* 📌 Mark tasks as **Active**, **Complete**, or **Incomplete**
* 🎨 Clean and responsive user interface
* ⚡ Dynamic DOM manipulation for seamless user interaction

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* DOM Manipulation
* Local Storage API

---

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/siddhantportfolio20/Task-Manager.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional installation or dependencies are required.
